      <h4 className="sb-section-item-heading">1. Function Statement or Function Declaration</h4>

            ```js
              function a() {
            ```

        <h4 className="sb-section-item-heading">2. Function Expression</h4>
            ```js
              var b = function() {
                console.log('Kiara');
              }
            ```

        <h4 className="sb-section-item-heading">3. Anonymous Function</h4>
        <p className="sb-section-item-paragraph">A function without name can not be called directly, it is used when passing a function as argument like in promises or loops etc...</p>
            ```js
           [1,2].map(function(item) {
            // anonymous function
           console.log(item);
           })
            ```

        <h4 className="sb-section-item-heading">4. Named Function Expresion</h4>
        <p className="sb-section-item-paragraph">Similar to function expression but the function also has a name. In below example you can call c() but b() will not work.</p>
            ```js
            var c = function b() {
                console.log('Named function expression');
            }
            ```
          <h4 className="sb-section-item-heading">5. Arrow Function</h4>
            ```js
            const arrow = () => {
                console.log('Arrow function');
            }
            ```
        <h4 className="sb-section-item-heading">6. Generator Function</h4>
        <p className="sb-section-item-paragraph">A <b>function*</b> declration creates generator function, calling this function will return Generator object. Note that insted of yield if we use return statement the next() will return object of 'returned value' and done: true.</p>
            ```js
            function* generatorFN() {
                            yield 1;
                            // if we put return here the {value: retured value, done: true}
                            yield 2;
                            yield 3;
                            // after the last yield next() will return {value: undefined, done:true}
                        }

                const itr = generatorFN();
                let v = itr.next();
                while (!v.done) {
                console.log('value = ', v.value);
                v = itr.next();
                ```

        <h4 className="sb-section-item-heading">7. Function Constructor or Constructor Function</h4>
        <p className="sb-section-item-paragraph"></p>
            ```js
            function Car(name) {
                this.carName = name;
                this.brand = 'Tata';
                this.printDetails = function () {
                    console.log(`The ${this.brand}'s ${this.carName} pretty safe!`);
            };
            }
            const myCar = new Car('Nexon');
            // calling Car() wihtout new keyword will return undefined unless we explicitly return something
            // similary if we put a return statement in between somewhere lets say before this.printDetails then the rest of the code is not included in newly created object(i.e printDetails is not avialable)
                myCar.printDetails();
                ```

        <h4 className="sb-section-item-heading">8. Getters & Setters</h4>
            ```js
                const personObj = {
                    firstName: '',
                    lastName: '',
                    //Note: getter and setter functions are used in object functions
                    get getFullName() {
                        return `${this.firstName} ${this.lastName}`;
                    },
                    set setFullName(name) {
                        //here you can do some sort of validation or sanitization etc
                        const tempFullName = name.split(' ');
                        this.firstName = tempFullName[0].toUpperCase();
                        this.lastName = tempFullName[1].toUpperCase();
                    },
                };
                personObj.setFullName = 'rowdy rathore';
                //personObj.firstName = 'abhilash' this will also work here
                console.log(personObj.getFullName);
                ```

        <h4 className="sb-section-item-heading">9. IIFE - Immediately Invoked Function Expression</h4>


                    }

                    <h4 className="sb-section-item-heading">Notes</h4>
